# Chemistry Notes 114-1
## Ch3 Stoichiometry
### 元素分析
1. 求 `O` 以外元素的質量百分比
2. 用 100% - 其他元素的質量百分比求 `O` 的質量百分比
3. 用 `質量百分比 / 原子量` 求莫耳比
4. 將所有係數同除最小係數
5. 同乘 n 使所有細數接近整數

## Ch4 Chemical Reactions
### Oxidation Number
1. `Element`: 0
2. `Ion`: Charge
4. `Cl`, `Br`, `I`: -1
5. `IA`: +1, `IIA`: +2

## Ch5 Gases
### 基本定律
1. 波以耳定律（Boyle's Law）: `PV = constant`
2. 查理定律（Charles's Law）: `V ∝ T`
3. 亞佛加厥定律（Avogadro's Law）: `V ∝ n`

### 理想氣體
$$PV = nRT$$
其中 `R ≈ 0.08206 L·atm/mol·K`

### 常用氣體條件
* STP：1 mol ≈ 22.4 L
* NTP（25°C）：1 mol ≈ 24.4 L

#### 分壓
$$P_{total} = P_1 + P_2 + ... + P_n = \frac{n_1RT}{V} + \frac{n_2RT}{V} + ... + \frac{n_nRT}{V}$$

## Ch6 Chemical Equilibrium
### 平衡常數
$$jA + kB ⇌ lC + mD$$
$$K = K_c = \frac{[C]^l [D]^m}{[A]^j [B]^k}$$

$$K_p = K_c(RT)^{\Delta n}$$
其中 `Δn = (l + m) - (j + k)`

$$Q = \frac{[C]^l [D]^m}{[A]^j [B]^k}$$
* `Q < K`: 向右
* `Q = K`: 平衡
* `Q > K`: 向左

$$P = \frac{n}{V}RT = cRT$$
其中 `c` 是濃度
