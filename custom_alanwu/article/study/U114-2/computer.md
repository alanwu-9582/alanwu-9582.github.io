# 人工智慧導論與計算機程式
## 生成式 AI 甲級題庫整理
### 生成式 AI 基礎概念
#### 定義
- 能從訓練資料中**生成新內容**
- 與傳統 AI（分析、分類）不同

#### 快速判斷
| 關鍵字 | 判斷 |
|--------|------|
| 生成新內容 | 正確 |
| 只做分析 | 錯誤 |


### 核心模型
#### 模型對應表
| 任務 | 模型 |
|------|------|
| 文字生成 | GPT |
| 圖像生成 | GAN / Diffusion |
| 語音生成 | Transformer |
| 圖像處理 | CNN |


#### GAN（生成對抗網路）
- 結構：Generator + Discriminator
- 核心：對抗學習


#### Diffusion Model
- 流程：加噪 → 去噪
- 關鍵：逐步還原圖像


#### Transformer
- 核心：Self-Attention
- 特點：
  - 處理長距離依賴
  - 可並行運算


#### BERT vs GPT
| 模型 | 特性 |
|------|------|
| BERT | 雙向理解 |
| GPT | 文本生成 |

### 偏見與公平性
#### 偏見來源
- 訓練資料不平衡

#### 偏見類型
| 類型 | 說明 |
|------|------|
| 抽樣偏見 | 資料不具代表性 |
| 表示偏見 | 某群體過多/過少 |
| 時間偏見 | 資料過時 |
| 語義偏見 | 語言關聯不平衡 |

#### 去偏技術
| 技術 | 說明 |
|------|------|
| Reweighing | 調整樣本權重 |
| Adversary | 對抗式辨識偏見 |
| Fair Representation | 去除敏感特徵 |
| Fair Intervention | 隱藏敏感屬性 |

#### 重要概念
| 名稱 | 說明 |
|------|------|
| Counterfactual Fairness | 比較不同屬性結果 |
| Algorithmic Discrimination | 對群體不公平 |
| Fairness Audit | 檢查偏見 |

### 重要題目

* 29：商業最常見應用是 自動化內容創作

* 33：2014 Ian Goodfellow 提出的是 GAN
* 38：長篇連貫文本表現出色的是 GPT-2

#### 機器學習
* 41：機器學習核心步驟之一是 數據收集
* 42：監督式學習 = 有標籤資料
* 46：避免神經元死亡問題是 Leaky ReLU
* 51：PCA = 降維
* 53：交叉驗證 = 評估泛化能力
* 56：Random Forest = 集成方法
* 58：Batch Normalization = 解決內部協變移

#### LLM / Transformer / 微調
* 64：題庫中最大參數量常考 PaLM
* 66：Transformer 訓練主要靠 反向傳播
* 72：Few-shot = 少量示例完成新任務
* 75：Multi-head Attention = 在不同子空間並行學習多種關係
* 77：RAG 的特徵「不包括 完全不依賴檢索」
* 80：Fine-tuning = 在任務資料上訓練 全部或部分權重
* 83：微調的優勢 不包括減少模型大小
* 88：GPT 主要用於 文本生成
* 91：保留預訓練權重只調少量新參數 = 凍結前層 + 適配層
* 92：LoRA 的低秩 = 高維權重矩陣分解成低秩矩陣
* 97：Adapter = 插入少量可訓練層
* 101：LLM 應用 不包括圖像識別

#### NLP 應用
* 104：完成特定任務的是 目標導向對話系統

* 112：詞轉向量常考 Word Embedding

#### 去偏、公平性、治理
* 130：減少偏見的方法，錯誤選項常是 增加模型參數
* 132：Counterfactual Fairness = 比較不同敏感屬性下的預測差異
* 134：對抗式去偏中負責辨別偏見的是 Adversary
* 137：Fair Intervention 常見做法 = 隱匿或屏蔽敏感屬性
* 140：保持下游表現最好的方式 = 先去偏再微調
* 144：假訊息大量生成與傳播的重要技術背景 = 深度學習
* 150：防止惡意使用 = 加強監管與合規
* 151：AI 道德準則中常見要求不包括獨立性
* 154：隱私保護技術通常不包括一般密碼雜湊
* 159：Algorithmic Discrimination = 對某些群體不平等對待
